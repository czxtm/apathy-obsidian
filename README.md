# Apathy — Obsidian Theme

A dark, sophisticated theme with deep purple-black backgrounds and vibrant neon-like accents. Part of the [Apathy theme family](https://github.com/czxtm/apathy-theme).






## Features

- **Deep dark background** — Rich purple-black tones (`#0C0C13`) easy on the eyes
- **Minted accent palette** — Soft cyan/mint highlights (`#D1F1EC`) for interactive elements
- **Carefully tuned callouts** — Every callout type has a distinct, harmonious color
- **Code block styling** — Syntax-aware colors matching the full Apathy palette
- **Metadata & UI polish** — Tabs, nav, titlebar, inputs all themed consistently

## Installation

### From Community Themes
1. Open **Settings → Appearance → Themes**
2. Click **Manage** and search for **Apathy**
3. Click **Install and use**

### Manual
1. Download `theme.css` and `manifest.json` from this repo
2. Create a folder called `Apathy` inside your vault's `.obsidian/themes/` directory
3. Place both files inside that folder
4. Go to **Settings → Appearance** and select **Apathy**
